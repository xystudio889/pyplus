## 描述

> 这个库需要python >=3.9,<3.12。推荐3.12。

对于 python 内容的扩展，它涵盖了科学、调试和其他领域。

---
## Version name rule
版本类型有 `r`,`pre`,`rc`。

版本类型 `r` 意思是发布版.他会存放正式版。

版本类型 `pre` 意思是预发布版，他会存放测试版，他会有较多的bug。

版本类型 `rc` 意思是发布版即将到来，他会存放晚期测试，并在不久后发布正式版。他会有较少的bug
