## 描述

> 这个库需要python >=3.8。

对于 python 内容的扩展，它涵盖了科学、调试和其他领域。

---

## 版本名规则
版本类型是 `dev`,`a`,`b`,`r`,`rc`,`post`.

版本类型`dev`意思是开发。他会存储开发板.有较多的bug。

版本类型`a`意思是alpha。他会存储早期测试版，有较多的bug.

版本类型`b`意思是beta。他会存储晚期测试。有较多的bug.

版本类型`rc`意思是预备发布版。正式版即将发布。他会有少量bug.

版本类型`post`意思是后发布。他会修复一些bug。

## 安装
运行`pip install python-plus-tools`

## 检查安装成功
安装完成后，打开python
```python
import pyplus
```
如果没有错误，安装成功
